# anime-search
Android application to find anime.

## Stack
- MVVM, single Activity.
- Migrated from RxJava to Coroutines
- Migrated from Dagger to Koin
- Retrofit
- Room
- Paging

## Features
- Top anime
- Anime filter
- Anime list paging
- StateView
